Configure adfsdiag tool in a new environment

Prerequisites and steps

  1. A Windows server machine (physical/virtual) joined to the domain where the target ADFS farm is installed. This machine will act as the web server and will house the ‘adfsdiag’ web tool.

  2. The computer account of the above-mentioned web server needs to be a local administrator on all target ADFS farm Web Application Proxy and ADFS servers.

  3. Install the following roles and features on the web server.



  1. Install Windows Identity Foundation 3.5. ( Important )



  1. Create an Application pool in IIS as follows





  1. Attach an SSL certificate with Default Web site.

  2. Copy the ‘adfsdiag’ web application folder to C:\inetpub\wwwroot on the web server. .\Publish folder contents may also be used.

  3. Expand “Default Web site” in IIS, right click the newly added ‘adfsdiag’ folder and choose “convert to application” , select the application pool created earlier ,click ok.



  1. If you receive error - Failed to update database "C:\INETPUB\WWWROOT\ADFSDIAG\APP_DATA\DATABASE.MDF" because the database is read-only while updating the vip table or farm name under configuration page, browse to C:\INETPUB\WWWROOT\ADFSDIAG\APP_DATA and provide IIS_ISURS(AppServerName\ IIS_ISURS) , modify permissions on the folder and if needed subsequent database and log files.



Appendix

  1. Current version is built using VS 2017. .Net framework 4.6.2.

  2. You may have to instate SNI wildcard bindings on the target Proxy or ADFS servers

To perform wildcard bindings , insert the binding as follows

netsh http add sslcert ipport=0.0.0.0:443 certhash=fcfc78505073ff9108e6a16e2115ef29255facb3 appid='{5d89a20c-beab-4389-9447-324788eb944a}' certstorename=MY

Where ‘fcfc78505073ff9108e6a16e2115ef29255facb3’ is the thumbprint of the ADFS Service Communication Certificate. 

and 

appid='{5d89a20c-beab-4389-9447-324788eb944a}' is the value of ‘appid’ returned by ‘netsh http show sslcert’ 



  1. The current version uses LocalDb. Please update connection strings as needed and SQL version.
